# Retail MD Analysis Skills

このドキュメントは、本プロジェクトにおけるデータ分析のロジックを定義した「スキル集」です。AIエージェントはこの基準に従って数値を解釈し、提案を行います。

## 1. ABC/XYZ分析スキル (Classification)
売上規模と需要の安定性を掛け合わせた在庫管理ロジック。

### ロジック定義
*   **ABCランク（売上規模）**: 累計売上構成比に基づき判定。
    - **A**: 上位70%（看板商品）
    - **B**: 70%〜90%（準主力）
    - **C**: 90%〜100%（不振品/新商品）
*   **XYZランク（需要の安定性）**: 日次販売数の変動係数（CV = 標準偏差 / 平均）に基づき判定。
    - **X (CV <= 0.5)**: 安定。予測が容易。
    - **Y (0.5 < CV <= 1.0)**: 普通。季節性や波がある。
    - **Z (CV > 1.0)**: 不規則。予測困難。

### 推奨アクション基準
- **AX/AY**: 自動補充。欠品厳禁。
- **AZ**: 予備在庫を厚めに持つ。
- **CZ**: 値下げ・撤退の優先検討。

## 2. WOS（在庫週数）管理スキル (Weeks of Supply)
現在の在庫が何週間で消化されるかを予測するスキル。

### 計算ロジック
- `WOS = 現在庫数 / 直近30日の平均週間販売数`
- 平均週間販売数 = (直近30日の総販売数 / 30) * 7

### アラート閾値
- **Red Alert (WOS < 2)**: 欠品間近。即時補充または店舗間移動が必要。
- **Yellow Alert (WOS > 20)**: 滞留リスク。プロモーションまたは価格調整が必要。

## 3. 時系列トレンド検知スキル (Trend Detection)
- **1Mトレンド**: 過去30日の日次推移を移動平均で追跡し、需要の「山」と「谷」を特定。
- **YoY（前年同月比）**: 昨年の同時期と比較し、成長率やカテゴリーシェアの変化を算出。
- **消化率分析**: 受入数量に対する累計販売数の割合をシーズン（春夏/秋冬）ごとに評価。
